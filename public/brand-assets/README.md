# Workshift Brand Assets

This package contains the official logos and brand assets for Workshift.

## Contents

- **/logos/logo-light.svg**: Full logo with wordmark for light backgrounds (vector).
- **/logos/logo-dark.svg**: Full logo with wordmark for dark backgrounds (vector).
- **/logos/logo-icon.svg**: Standalone brand mark (vector).
- **/png/logo-light.png**: Full logo for light backgrounds (1200x400 PNG).
- **/png/logo-dark.png**: Full logo for dark backgrounds (1200x400 PNG).
- **/png/logo-icon.png**: Brand mark (512x512 PNG).

## Brand Colors

- **Lime**: #9CE069 (Primary Accent)
- **Sage**: #E6E8DD (Base Background)
- **Dark**: #000000 (Primary Text/Background)

## Typography

- **Primary Font**: Inter (700 Bold for wordmark)
